## Custom styling css

[accordion style="color: blue;"]

<div class='text-lg font-semibold text-white'>Style eksempel</div>
<ul>
  <li>color: blue; = blå tekst farge</li>
  <li>color: #ffffff; = hvit tekst farge</li>
  <li>padding: 5px; = padding 5px</li>
</ul>

<div class='text-lg font-semibold text-white'>Additional css eksempel</div>
<div>
  Bruk css styling for å endre på selve accordion konteineren, du kan bruke
  additional css for å endre på shortcoden med bruk av data-name="accordion"
  <ul>
    <li>data-name="accordion" = Kontainer</li>
    <li>data-name="accordion" > summary = Tittel</li>
    <li>data-name="accordion" > summary > = Pil / arrow</li>
    <li>data-name="accordion" > div = tekst</li>
  </ul>
</div>
